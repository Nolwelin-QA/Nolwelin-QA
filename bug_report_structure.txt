Основные поля
1. ID — уникальный идентификатор бага
2. Заголовок (Title)
3. Шаги воспроизведения (STR)
4. Ожидаемый результат (Expected result)
5. Фактический результат (Actual result)
6. Серьезность (Severity)

Дополнительные поля
1. Краткое описание (Summary) 
2. Скриншот, видео (Attachments)
3. Ссылка на требование (Requirement) 
4. Приоритет (Priority)
5. Окружение (Environment)
6. Статус (Status)
7. Автор (Author, Reporter)
8. Исполнитель (Assignee)
9. Комментарии (Comments)
10. Версия (Version in which the bug was found)
11. Версия исправления (Version to which the bug fix will be added)

1. STR

V. BUG TRACKING:
1. Severity
